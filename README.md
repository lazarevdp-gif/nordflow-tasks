# NordFlow Tasks

Внутренний трекер задач для компании NordFlow.

## Запуск

### Требования
- Python 3.10+

### Установка

```bash
git clone https://github.com/lazarevdp-gif/nordflow-tasks.git
cd nordflow-tasks
pip install -r requirements.txt
```

### Запуск сервера

```bash
cd server
python app.py
```

Откройте в браузере: http://localhost:5000
