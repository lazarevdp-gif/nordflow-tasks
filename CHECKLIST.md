# Чек-лист тестирования NordFlow Tasks

## 1. Авторизация

- [ ] Вход с верными данными (dmitriy.lazarev@nordflow.ru / password123)
- [ ] Вход с неверным паролем → ошибка 401
- [ ] Вход с несуществующим email → ошибка 401
- [ ] Пустой email/пароль → ошибка 400
- [ ] Rate limiting: 5+ неудачных попыток → 429 "Слишком много попыток"
- [ ] Logout → сессия удаляется, запрос к /api/auth/me возвращает null
- [ ] Запрос без сессии → 401 на всех защищённых эндпоинтах
- [ ] Сессия истекает через 8 часов неактивности

## 2. Регистрация / Управление пользователями

- [ ] Admin: создание пользователя (все поля, валидный email)
- [ ] Admin: создание с коротким паролем (<6 символов) → ошибка 400
- [ ] Admin: создание с дублем email → ошибка 400
- [ ] Admin: создание без обязательных полей → ошибка 400
- [ ] Employee: попытка создать пользователя → 403 Forbidden
- [ ] Admin: редактирование профиля другого пользователя
- [ ] Admin: смена роли пользователя
- [ ] Admin: смена пароля пользователя
- [ ] Employee: редактирование чужого профиля → 403
- [ ] Employee: редактирование своего профиля (имя, позиция, фото, тема)
- [ ] Смена пароля на <6 символов → ошибка 400
- [ ] Admin: удаление пользователя
- [ ] Non-admin: удаление пользователя → 403

## 3. Проекты

- [ ] Загрузка списка проектов (admin видит все, PM/employee — только свои)
- [ ] Создание проекта (PM/admin)
- [ ] Employee: попытка создать проект → 403
- [ ] Редактирование проекта (название, описание, статус, даты)
- [ ] Управление участниками проекта (добавление/удаление)
- [ ] Детальная страница проекта: инфо + список задач + участники
- [ ] Проект с задачами: корректный подсчёт open_tasks

## 4. Задачи

- [ ] Список задач с фильтрами (проект, статус, приоритет, исполнитель)
- [ ] Поиск задач по названию/идентификатору
- [ ] Создание задачи (название обязательно, макс 200 символов)
- [ ] Создание задачи без названия → ошибка 400
- [ ] Создание задачи с project_id (привязка к проекту)
- [ ] Создание задачи без project_id (standalone)
- [ ] Уведомление исполнителю при назначении задачи
- [ ] Редактирование задачи (только owner, PM или admin)
- [ ] Employee: редактирование чужой задачи → 403
- [ ] Смена статуса (только owner, PM или admin)
- [ ] Смена на недопустимый статус → ошибка 400
- [ ] История изменений задачи (field, old_value, new_value)
- [ ] Уведомление при смене статуса/дедлайна
- [ ] Удаление задачи (только creator или admin)
- [ ] Employee: удаление чужой задачи → 403
- [ ] Детальная страница задачи: инфо + комментарии + вложения + история

## 5. Комментарии

- [ ] Добавление комментария к задаче
- [ ] Уведомление исполнителю при новом комментарии
- [ ] Редактирование своего комментария
- [ ] Редактирование чужого комментария → 403 (кроме admin)
- [ ] Удаление своего комментария
- [ ] Удаление чужого комментария → 403 (кроме admin)

## 6. Уведомления

- [ ] Список уведомлений (свои, лимит 50)
- [ ] Счётчик непрочитанных
- [ ] Отметка всех как прочитанных
- [ ] Отметка одного как прочитанного
- [ ] Уведомления не приходят самому себе

## 7. Файлы

- [ ] Загрузка файла к задаче (whitelist: jpg, png, pdf, doc...)
- [ ] Загрузка запрещённого типа файла → ошибка 400
- [ ] Загрузка файла >10MB → ошибка 400
- [ ] Загрузка аватара (только изображения)
- [ ] Загрузка аватара非图片 → ошибка 400
- [ ] Корректное отображение аватара в профиле и команде

## 8. Дашборд

- [ ] Статистика: active_tasks, overdue_tasks, due_this_week
- [ ] Ближайшие дедлайны (sorted by deadline)
- [ ] Последние обновления задач
- [ ] Мои проекты со счётчиками
- [ ] Team stats для admin/PM

## 9. Команда

- [ ] Список сотрудников с active/overdue задачами
- [ ] Переход в профиль сотрудника
- [ ] Профиль сотрудника: аватар, ФИО, позиция, роль, email, задачи

## 10. Поиск

- [ ] Поиск по задачам (название, идентификатор)
- [ ] Поиск по проектам (только доступные)
- [ ] Поиск по сотрудникам (ФИО)
- [ ] Минимум 2 символа для поиска

## 11. Безопасность

- [ ] Security headers: X-Content-Type-Options, X-Frame-Options, X-XSS-Protection, Referrer-Policy
- [ ] CORS: запросы с другого origin блокируются
- [ ] Cookie: HttpOnly, SameSite=Strict
- [ ] Session timeout 8 часов
- [ ] Debug mode выключен
- [ ] Password hashing: pbkdf2 с солью
- [ ] Старые хеши (SHA-256) работают обратно совместимо
- [ ] SQL-инъекции: параметризованные запросы
- [ ] XSS: пользовательский ввод экранируется через textContent

## 12. Граничные случаи

- [ ] Задача без дедлайна
- [ ] Задача без исполнителя
- [ ] Задача без проекта
- [ ] Проект без участников
- [ ] Двойной submit формы
- [ ] Параллельные запросы
- [ ] Пустой тело запроса (null body)
- [ ] Невалидный JSON в теле запроса
